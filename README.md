# 🐱 CatFacts

A fun frontend project that displays **random cat facts** using HTML, CSS, and JavaScript.  
It fetches data from the [Cat Facts API](https://catfact.ninja/fact) to show new facts dynamically.

---

## 🚀 Features
- Fetches random cat facts from an API
- Clean and responsive UI built with HTML & CSS
- Interactive button to load new facts
- Simple JavaScript logic with `fetch()` for API calls
